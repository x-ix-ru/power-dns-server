# Спецификация задачи 001: Обновить pdns.conf

## Описание

Заменить значения по умолчанию в `pdns.conf` на целевые для чистого образа.

## Полный контекст

Текущий `pdns.conf` содержит:
- `api-key=changeme` → нужно `api-key=secret`
- `webserver-allow-from=0.0.0.0/0,::/0` → оставить `0.0.0.0/0,::/0` (уже верно)
- Остальные директивы актуальны, но можно привести к чистому виду

Требования задачи:
1. `api-key=secret` — default API key
2. `webserver-allow-from=0.0.0.0/0` — разрешить доступ со всех адресов
3. Остальные настройки оставить как есть (sqlite3 backend, порт 53, daemon=no)

## Технические детали

### Изменения в `pdns.conf`

- Строка 23: `api-key=changeme` → `api-key=secret`
- Строка 26: `webserver-allow-from=0.0.0.0/0,::/0` — уже корректно, оставить
- Остальные секции без изменений

### Конечное содержание ключевых директив

```
api=yes
api-key=secret
webserver-address=0.0.0.0
webserver-port=80
webserver-allow-from=0.0.0.0/0,::/0
```

- [ ] Все остальные директивы сохранены и корректны
- [ ] Формат файла не повреждён (секции, комментарии на месте)
