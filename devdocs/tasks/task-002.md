# Задача 002: Оптимизировать Dockerfile

## Описание

Оптимизировать `Dockerfile` для получения чистого, лёгкого образа PowerDNS:
- Слить `apk add` команды в один слой
- Создать `.dockerignore` для исключения `.git/`, `devdocs/`, `*.md` и др.
- Убедиться, что образ минимальный и не содержит лишних пакетов

## Покрытие требований

- **spec.md §3.2** — Оптимизации образа: минимизация слоёв, .dockerignore
- **spec.md §4.1** — Размер образа ≤ 30 MB
- **Задача:** Clean, lightweight, fast Docker image
